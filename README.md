# JavaWeb_BookCity

+ 相关技术
	+ 前端：html，css，js，jQuery，jsp，EL，JSTL；
	+ 后端：servlet，filter请求拦截，JDBC
	+ 数据库：MySQL
	+ 服务器：tomcat
